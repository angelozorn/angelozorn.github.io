Project highlights

Selected work in machine learning, modeling, and data.

Scissor Kick Explorer — MLB batting stance analysis

Interactive Streamlit app comparing scissor-kick vs non-scissor batting mechanics across MLB hitters. Explore group-level aggregates and per-player stats including swing metrics, batted-ball data, and a virtual batter's-box stance plot.

Live app

Rest vs Rust in the NFL Playoffs

Data dashboard exploring whether bye-week rest causes rust in NFL playoff outcomes. Analyzes Super Bowl rates, divisional round performance, roster rest flags, injury impacts, and closing-line results from 2002–2023.

Live dashboard

Deep learning binary classification financial forecasting

A PyTorch deep neural network (four hidden layers) to find nonlinear patterns in market data. Features are OHLCV for a ticker (e.g. SPY); the model predicts next-day direction with reported accuracy, recall, and precision.

GitHub

Modeling asymptomatic regional spread of COVID-19

Regional spread driven by asymptomatic contact rates, using ODEs and an SAIRD-style compartment model with a mobility component. The project poster summarizes methods and results.
